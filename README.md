## Question

Do more hawkish FOMC statements coincide with higher short-term Treasury yield, a stronger dollar, and weaker equity returns?


## Data

- FOMC statements
- daily asset price data


## Method

- scraping statements
- cleaning text
- creating tone score
- matching statements dates with market data
- running basic regressions
- producing plots and interpretation


## Outputs

- tone score distribution
- tone over time
- tone vs yield change
- tone vs S&P return
- tone vs USD change
- regression table
